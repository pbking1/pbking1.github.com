C++ Reference and Pointer

what is pointer

  • 形参
    • formal parameter
    • 定义时函数里面的变量
      • e.g int swao(int a, int b)
      • a和b就是形参
  • 实参
    • actual parameter
    • 在主函数里面引用的
      • e.g swap(x,y)
      • x和y就是实参，实际的值

注意事项

  • 在可以使用引用的情况下，不要用指针
  • 引用不能为空，当对象为NULL时，必须使用指针
  • 引用不允许重新赋值，当使用一个变量指向不同的对象的时候，必须使用指针
